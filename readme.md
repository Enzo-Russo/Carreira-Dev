# Estudos em Python e SQL

Repositório de prática e aprendizado, com exercícios e mini-projetos em **Python** e **Banco de Dados (SQL/MySQL)**.

## Pastas

- `01-python-fundamentos/` → exercícios e fundamentos (tabuada, etc.)
- `02-sql/` → estudos de SQL/MySQL (CRUD e conexão)
- `03-mini-projetos/` → mini-sistemas (estoque, vendas, loja, etc.)
- `data/` → arquivos de dados (ex.: CSV)
- `99-rascunhos/` → versões antigas e testes

## Como executar

Rode qualquer arquivo com:

```bash
python caminho/do_arquivo.py
```
